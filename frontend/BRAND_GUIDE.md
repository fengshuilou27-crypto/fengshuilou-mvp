# 風水樓品牌視覺設計系統 v3.0

## 品牌定位

**定位**：將傳統風水學與現代 AI 技術結合的樓盤匹配平台
**目標用戶**：25-50 歲香港置業者，重視傳統文化但習慣數碼工具
**品牌個性**：專業、可信、傳統、現代、吉祥

---

## 色彩系統

### 主色 — 深翡翠綠
```
#1a3c2a  (主色 500)
```
- 代表風水、自然、生命力、穩重
- 用於：Header、Footer、主要按鈕、標題

### 輔色 — 古金色
```
#c9a961  (輔色 500)
```
- 代表吉祥、傳統、尊貴、財富
- 用於：強調文字、CTA 按鈕、裝飾元素、徽章

### 背景色 — 宣紙白
```
#f5f0e8
```
- 代表傳統書卷、文化底蘊
- 用於：頁面背景、卡片背景

### 文字色 — 墨黑
```
#1a1a2e
```
- 古典韻味，比純黑更柔和
- 用於：正文文字

### 功能色
```
成功：#2d6a4f (翡翠綠)
警告：#d4a017 (琥珀金)
危險：#c41e3a (吉祥紅)
信息：#4a90d9 (青花藍)
```

## 視覺風格

### 新中式現代風格
- 結合傳統風水元素與現代 UI 設計
- 避免過度中式（不適用龍鳳、太極等圖案）
- 提煉傳統元素的抽象形式（如網格紋理、金色點綴）

### 卡片設計
- 大圓角（16-20px）
- 柔和陰影（基於品牌色，不純黑）
- 白色背景，細邊框

### 按鈕設計
- 主按鈕：深綠色 + 金色 hover
- CTA 按鈕：金色漸變 + 光暈效果
- 次按鈕：白色 + 綠色邊框

### 裝飾元素
- 金色分隔線（漸變）
- 網格紋理背景（低透明度）
- 圓形光暈（模糊）
- 波浪形分隔（SVG）

## Logo 設計概念

### 標誌（Symbol）
- 簡化的「風」字或「羅盤」抽象圖形
- 深綠色底 + 金色線條
- 圓角方形或圓形外框

### 文字標誌（Wordmark）
- 「風水樓」— 思源宋體，粗體
- 「FengShuiLou」— 無襯線體，全大寫，字距加寬

### 組合方式
- 圖形 + 中文（主要 Logo）
- 圖形 + 英文（次級 Logo）
- 僅圖形（Favicon、小尺寸）

---

## 動效設計

### 頁面載入
- 元素漸入 + 向上滑動（fade-in + translateY）
- 錯開時序（stagger 0.1-0.15s）

### 滾動動畫
- 使用 Framer Motion 的 `whileInView`
- 元素進入視口時觸發動畫
- 僅觸發一次（`once: true`）

### 微交互
- 按鈕 hover：輕微上移 + 陰影增強
- 卡片 hover：輕微上移 + 陰影增強
- 連結 hover：顏色變為金色

## 設計原則

1. **傳統與現代平衡**：傳統色彩 + 現代排版
2. **信任感**：專業字體、穩重色彩、清晰結構
3. **吉祥感**：金色點綴、溫暖色調、圓潤形狀
4. **可讀性**：足夠對比度、適當行高、清晰層級
5. **一致性**：全局統一色彩、字體、圓角、陰影
